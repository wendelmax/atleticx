# Documentacao do produto

Este diretorio consolida a idealizacao do sistema mobile e web para gamificacao em artes marciais.

## Arquivos

- `sistema-gamificacao-artes-marciais.md`: visao de produto, dominios, fluxos, arquitetura, seguranca e monetizacao.
- `backlog-mvp.md`: epicos, historias e plano de entregas por fase.
- `decisao-marca-atleticx.md`: decisao oficial de naming, assinatura e diretrizes de uso da marca.
- `arquitetura-tecnica.md`: arquitetura de software, colecoes, acessos e fluxo de token.
- `setup-projeto.md`: pre-requisitos, configuracao e execucao do projeto.
